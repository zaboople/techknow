# Editor / Console font
# you do not get a separate item for console font
Intellij Idea -> Settings -> Editor -> Font

# Grep files
ALT+SHIFT+F

# Go to file:
ALT+SHIFT+O

# Avoid jumping into crap reflection and what have you during debugging:
Intellij -> Settings -> search on "Stepping" -> add package names you don't want to step into, esp. "org.springframework.aop.*"
